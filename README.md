### Simple DLL injector that saves multiple DLLs so you don't have to search for them every time.

- Modern dark UI.
- Simple to use.
- Single .exe
- Doesn't require admin rights.
- Fixes DLL permissions before injecting.

(x64 release build only. for x86 support build it yourself)
